## 🌟 简介

**无知之海** 是一个将北京大学社会学考研知识点完全游戏化的剧情RPG学习网站。

在这里，你将化身为「社会学徒」，穿越古典理论的废墟、踏过现代理论的迷宫、在方法论的田野中寻找真相的碎片，最终攀登「社会学深渊」的最高峰。

### 核心特色

- 📚 **完整考点覆盖** - 完全适配北京大学社会学考研全套考点
- 🎮 **剧情RPG体验** - 沉浸式世界观，角色扮演+主线剧情
- 🧩 **解谜闯关体系** - 知识连线、拼图解谜、情境模拟
- 📊 **进度可视化** - 100天循序渐进，章节锁卡+成就系统
- 💾 **自动存档** - 浏览器本地存储，随时继续学习
- 📱 **多端适配** - 支持桌面和移动端

## 🚀 快速开始

### 在线访问

直接访问 GitHub Pages 地址：
```
https://yourusername.github.io/pku-sociology-rpg
```

### 本地部署

1. **克隆仓库**
```bash
git clone https://github.com/yourusername/pku-sociology-rpg.git
cd pku-sociology-rpg
```

2. **使用 GitHub Pages**
   - Fork 此仓库
   - 在 Settings > Pages 中启用 GitHub Pages
   - 选择 `main` 分支和 `/ (root)` 目录

3. **本地预览**
   - 使用 VS Code 的 Live Server 插件
   - 或使用 Python：`python -m http.server 8000`
   - 或使用 Node.js：`npx serve`

## 📖 内容大纲

### 100天闯关体系

| 阶段 | 天数 | 主要内容 |
|------|------|----------|
| 觉醒期 | Day 1-10 | 社会学入门、核心概念 |
| 古典之旅 | Day 11-30 | 涂尔干、韦伯、马克思 |
| 现代转型 | Day 31-50 | 结构功能主义、符号互动论 |
| 批判与反思 | Day 51-60 | 法兰克福学派、后现代理论 |
| 方法入门 | Day 61-70 | 社会研究方法 |
| 统计进阶 | Day 71-80 | 社会统计学 |
| 分支探索 | Day 81-90 | 城市、劳动、人口社会学 |
| 真题冲刺 | Day 91-100 | 全真模拟与真题演练 |

### 世界观设定

- **世界观名称**: 「无知之海」与「社会学深渊」
- **核心概念**: 无知之雾、社会学深渊、社会角色碎片等
- **主角设定**: 社会学徒（可自定义昵称）
- **NPC设定**: 涂尔干先生、韦伯学者、孔德老人等

## 🎮 游戏机制

### 经验值与等级

- 完成章节/答题获得经验值
- 升级解锁新的章节和功能
- 100级成长体系

### 成就系统

- 新手成就、进阶成就、特殊成就
- 完成特定条件解锁成就
- 成就奖励额外经验值

### 章节锁卡

- 需要达到特定等级才能解锁后续章节
- 完成前置章节自动解锁
- 章节进度自动保存
