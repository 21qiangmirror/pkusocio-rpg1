## 1.2 主线剧情设定

### 📜 故事背景

```
═══════════════════════════════════════════════════════════════════
                           【序章：迷雾中的觉醒】
═══════════════════════════════════════════════════════════════════

你是一名普通的大四学生，正站在人生的十字路口。
面对就业压力、身份焦虑和对"社会到底是怎么运作的"的好奇，
你在图书馆角落发现了一本泛黄的旧书——

《社会学入门：从入门到放弃？》

当你翻开扉页的瞬间，文字化作金色光芒将你吞噬...
醒来时，你发现自己身处一片名为「无知之海」的迷雾之中。
远处，一座名为「社会学深渊」的巨塔刺破天际，
那是所有社会学知识汇聚的地方，也是你此行的终点。

一个苍老的声音在你耳边响起：

「年轻人，你将在这100天的试炼中，
  穿越古典理论的废墟、踏过现代理论的迷宫、
  在方法论的田野中寻找真相的碎片、
  最终攀登社会学深渊的最高峰——」

「去吧，当你集齐所有知识碎片的那天，
  你将获得真正的社会学之眼，
  看清这个世界的真实模样。」

═══════════════════════════════════════════════════════════════════
```

| 第九卷 | 《社会的百态》 | 分支学科探索 | 城市、劳动、人口、心理社会学 |
| 第十卷 | 《深渊之巅》 | 综合训练与真题挑战 | 真题演练、综合应用 |

### js/core/game-engine.js - 游戏引擎核心

```javascript
/**
 * 游戏引擎核心文件
 * 负责游戏状态管理、进度追踪、存档系统等核心功能
 */

// ==================== 游戏状态管理 ====================

class GameState {
    constructor() {
        this.playerData = this.loadPlayerData();
        this.chapterProgress = this.loadChapterProgress();
        this.settings = this.loadSettings();
    }
    
    // 初始化玩家数据
    initPlayer(name) {
        return {
            id: this.generateUUID(),
            name: name,
            level: 1,
            experience: 0,
            attributes: {
                theory: 0,
                methods: 0,
                statistics: 0,
                branches: 0
            },
            inventory: [],
            achievements: [],
            unlockedNPCs: [],
            currentChapter: 'prelude',
            currentDay: 1,
            totalDaysCompleted: 0
        };
    }
    
    // 存档管理
    save() {
        localStorage.setItem('pku-sociology-rpg-save', JSON.stringify({
            playerData: this.playerData,
            chapterProgress: this.chapterProgress,
            timestamp: Date.now()
        }));
    }
    
    load() {
        const saved = localStorage.getItem('pku-sociology-rpg-save');
        return saved ? JSON.parse(saved) : null;
    }
}

// ==================== 经验值与等级系统 ====================

class ExperienceSystem {
    static XP_TABLE = {
        1: 0, 2: 100, 3: 250, 4: 450, 5: 700,
        6: 1000, 7: 1350, 8: 1750, 9: 2200, 10: 2700,
        // ... 更多等级
    };
    
    // 完成任务获得经验
    static awardXP(amount, type = 'general') {
        const player = gameState.playerData;
        player.experience += amount;
        
        // 更新属性
        if (type in player.attributes) {
            player.attributes[type] += Math.floor(amount * 0.5);
        }
        
        // 检查升级
        this.checkLevelUp();
        
        // 触发UI更新
        uiManager.updateExperienceBar();
    }
    
    // 升级检查
    static checkLevelUp() {
        const currentLevel = gameState.playerData.level;
        const currentXP = gameState.playerData.experience;
        
        for (let level = currentLevel + 1; level <= 100; level++) {
            if (currentXP >= this.XP_TABLE[level]) {
                gameState.playerData.level = level;
                this.onLevelUp(level);
            }
        }
    }
}

// ==================== 章节解锁系统 ====================

class ChapterUnlocks {
    static CHAPTER_REQUIREMENTS = {
        'prelude': { level: 1, prerequisites: [] },
        'volume1': { level: 5, prerequisites: ['prelude'] },
        'volume2': { level: 15, prerequisites: ['volume1'] },
        'volume3': { level: 30, prerequisites: ['volume2'] },
        'volume4': { level: 45, prerequisites: ['volume3'] },
        'volume5': { level: 60, prerequisites: ['volume4'] },
        'volume6': { level: 75, prerequisites: ['volume5'] },
        'finale': { level: 90, prerequisites: ['volume6'] }
    };
    
    static checkUnlock(chapterId) {
        const req = this.CHAPTER_REQUIREMENTS[chapterId];
        const player = gameState.playerData;
        
        // 检查等级
        if (player.level < req.level) return false;
        
        // 检查前置章节
        for (const prereq of req.prerequisites) {
            if (!this.isChapterCompleted(prereq)) return false;
        }
        
        return true;
    }
    
    static isChapterCompleted(chapterId) {
        return gameState.chapterProgress[chapterId]?.completed || false;
    }
}

// ==================== 游戏引擎导出 ====================

const gameEngine = {
    state: new GameState(),
    xp: ExperienceSystem,
    chapters: ChapterUnlocks,
    
    init() {
        // 加载存档或创建新游戏
        const savedData = this.state.load();
        if (savedData) {
            this.state.playerData = savedData.playerData;
            this.state.chapterProgress = savedData.chapterProgress;
        }
        
        // 初始化UI
        uiManager.init();
        
        // 显示首页
        this.loadPage('home');
    }
};

export default gameEngine;
```

| 模块 | 推荐技术 | 理由 |
|-----|---------|-----|
| 前端框架 | Vanilla JS / Vue 3 | 轻量、SEO友好 |
| 样式方案 | CSS Variables + BEM | 主题切换、模块化 |

| 音效 | Howler.js | 跨浏览器音频 |
| 数据存储 | localStorage + IndexedDB | 本地持久化 |
| 构建工具 | Vite | 快速开发 |
