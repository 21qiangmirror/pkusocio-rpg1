## 快速开发指南

### 1. 克隆项目
```bash
git clone https://github.com/yourusername/pku-sociology-rpg.git
cd pku-sociology-rpg
```

### 2. 创建新章节
参考 `chapters/prelude/day01.html` 的模板结构创建新章节。

### 3. 添加题目
在 `data/questions/` 目录下的JSON文件中添加新题目。

### 4. 添加NPC对话
在 `data/npcs/` 目录下的JSON文件中添加NPC对话。

### 5. 部署到GitHub Pages
```bash
npm install
npm run deploy
```
